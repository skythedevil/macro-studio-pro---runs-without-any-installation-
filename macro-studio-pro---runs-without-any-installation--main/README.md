# 🚀 Testing Macro Studio Pro

**Advanced Productivity & Automation Suite** 

Testing Macro Studio Pro is a professional-grade, portable automation tool designed for Windows. It allows users to simulate high-performance mouse, keyboard, and scroll interactions using a sleek, logical 3-column "Studio" layout.

Built entirely in PowerShell, it requires **zero dependencies** and **no administrative privileges**, making it the perfect solution for restricted corporate environments and developers looking for lightweight automation.

---

## ✨ Key Capabilities

- **🖥️ Studio Layout**: A refined 3-column dashboard separating your Macro Engine, Project Operations, and Precise Parameters.
- **🎨 Multi-Theme Support**: Instantly switch between a professional **Dark Mode** and a clean **Facebook-Style Light Mode**.
- **🧠 Visual AI Matching**: Uses 60x60 pixel anchor matching to auto-align clicks even if windows have shifted on the screen.
- **🖼️ Visual Anchor Previews**: Real-time image previews for every step in your macro sequence.
- **🪟 Smart Window Context**: Automatically records and waits for specific application windows before executing actions.
- **✏️ In-GUI Editing**: Modify text, scroll units, or wait delays directly from the dashboard without re-recording.
- **⌨️ Global Hotkeys**: 
  - `F8`: Start/Stop Recording or Playback.
  - `F9`: Pause/Resume Session instantly.
- **🛡️ AV Hardened**: Professional metadata and refactored native methods to minimize false-positive antivirus detections.

## 🚀 Getting Started

1. **Download**: Clone this repository or download [TestingMacro.ps1](TestingMacro.ps1).

## 🚀 How to Run
1.  **Download** the repository as a ZIP and extract it.
2.  **Right-click** `TestingMacro.ps1` -> Properties -> Check **Unblock** -> OK (Required for Internet-downloaded scripts).
3.  **Double-click `RunStudio.bat`** to start the application instantly.

> [!TIP]
> Using the `.bat` launcher automatically bypasses PowerShell security policies, ensuring the program runs without any complex setup.

3. **Record**: Click **RECORD SESSION (F8)** and perform your task.
4. **Automate**: Save your project as a `.json` file and replay it anytime with **PLAY MACRO**.
